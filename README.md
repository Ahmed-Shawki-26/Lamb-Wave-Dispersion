# Lamb Wave Dispersion

Python tools for Lamb wave dispersion curves and particle displacement in isotropic plates (phase/group velocity, wave structure, animations).

## Install

```bash
pip install -r requirements.txt
```

## Run

- **GUI:** `python lamb_wave_gui.py` (or `run_gui.bat` on Windows)
- **Scripts:** `python dispersion_analysis.py`, `python animate_lamb_wave.py`

Outputs go to `results/`.
